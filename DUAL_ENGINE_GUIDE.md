# 双引擎调度架构使用指南

## 📋 概述

v5.0版本引入了**双引擎调度架构**，从单一的加权投票升级为均值回归引擎和趋势引擎并行工作，解决了之前推荐结果100%为超跌股的问题。

## 🎯 核心设计

### 1. 双引擎架构

```
全市场股票 (808只)
    ↓
[策略计算]
├── 均值回归引擎 → mr_score (PB/PE/PEPB/DUAL/BOLL/RSI/KDJ)
└── 趋势引擎 → trend_score, momentum_score (Trend_Composite)
    ↓
[软过滤]：adjusted_score = mr_score × (0.5 + 0.5×trend_weight)
    ↓
[市场状态判断]：指数MA20/MA60斜率 → 趋势市/震荡市
    ↓
[双榜单生成]
├── 超跌榜：按adjusted_score排序取N1
└── 趋势榜：按trend_rank_score排序，去重后取N2
    ↓
[合并推荐]：超跌榜 + 趋势榜 = TOP20
```

### 2. 关键参数

```python
# 均值回归引擎权重（基于836只股票回测优化）
MR_WEIGHTS = {
    'PB': 2.0, 'PE': 1.68, 'PEPB': 1.61, 'DUAL': 1.39,
    'BOLL': 1.95, 'RSI': 1.82, 'KDJ': 1.5,
    'MACD': 1.13, 'MA': 0.88,
    'NEWS': 0.32, 'SENTIMENT': 0.32, 'MONEY_FLOW': 0.3,
}

# 趋势引擎配置
TREND_WEIGHT_BASE = 0.5       # 基础权重（无趋势时的最低权重）
TREND_WEIGHT_RANGE = 0.5      # 权重调整范围（趋势越强，权重越高）
TREND_SCORE_WEIGHT = 0.7      # 趋势得分权重
MOMENTUM_SCORE_WEIGHT = 0.3   # 动量得分权重
TREND_STRENGTH_THRESHOLD = 0.03  # 市场趋势强度阈值（MA20/MA60斜率）
```

### 3. 市场状态判断

| 市场状态 | 判断条件 | 超跌榜数量 | 趋势榜数量 |
|---------|---------|-----------|-----------|
| 趋势市 | MA20/MA60斜率 > 3% | 10只 | 10只 |
| 震荡市 | MA20/MA60斜率 ≤ 3% | 15只 | 5只 |

## 🔧 参数调优

### 1. 调整双榜单比例

如果发现超跌股过多，可以修改 `recommend_today.py` 中的榜单数量：

```python
# 原配置（震荡市）
mr_n, trend_n = (15, 5)

# 调整为更平衡的配置
mr_n, trend_n = (12, 8)
```

### 2. 调整趋势强度阈值

如果市场状态判断不准确，可以调整阈值：

```python
# 原配置
TREND_STRENGTH_THRESHOLD = 0.03  # 3%

# 更宽松（更容易判定为趋势市）

# 更严格（更倾向判定为震荡市）

```

### 3. 调整软过滤权重

如果想让趋势对超跌股的影响更大：

```python

# 增大趋势影响（让强趋势股更突出）
TREND_WEIGHT_BASE = 0.3

```

## 📊 趋势策略详解

### Trend_Composite（复合趋势因子）

融合三个维度：

1. **ADX趋势强度**：
   - ADX > 20：趋势强
   - ADX ≤ 20：震荡市
   - 输出：adx_strength [0, 1]

2. **均线排列**：
   - MA5 > MA10 > MA20 > MA60：多头排列（+1）
   - MA5 < MA10 < MA20 < MA60：空头排列（-1）
   - 输出：alignment_factor [-1, 1]

3. **波动率调整动量**：
   - 20日涨跌幅 / ATR（波动率标准化）
   - EWM标准化后归一化到 [-1, 1]
   - 输出：momentum_factor [-1, 1]

**最终得分**：
```python
composite = (alignment_weight × alignment_factor + momentum_weight × momentum_factor) × adx_strength
```

### 信号判断

- `composite > 0.6`：BUY（强趋势向上）
- `composite < -0.6`：SELL（强趋势向下）
- `-0.6 ≤ composite ≤ 0.6`：HOLD（震荡或弱趋势）

## 📊 持仓分析逻辑（v5.0.1）

### 策略：均值回归为主 + 趋势风险修正

**基础判断**（均值回归引擎）：
- score < -5 或 sell_cnt ≥ 3 → 🔴 建议减仓/止损
- score < 0 或 sell_cnt ≥ 2 → 🟡 关注风险
- score > 5 且 buy_cnt ≥ 3 → 🟢 可以加仓
- buy_cnt ≥ 2 → 🟢 继续持有

**趋势风险修正**：
- **趋势转弱**（trend_score < -0.3）：
  - 🟢 可以加仓 → 🟡 可持有，但暂缓加仓
  - 🟢 继续持有 → 🟡 谨慎持有
- **趋势向好**（trend_score > 0.3）：
  - 🟡 关注风险 → 🟡 可继续观察（趋势尚可）
  - 🟢 继续持有 → 🟢 继续持有（趋势健康）

## 🌟 双优股票机制（v5.0.2）

### 什么是双优股票？

**双优股票**是指同时满足以下两个条件的股票：
1. **进入超跌榜**：`mr_score`（均值回归得分）经过趋势软过滤后位列前茅，具备低估值反弹潜力
2. **进入趋势榜**：`trend_rank_score`（趋势质量得分）位列前茅，具备强劲上升趋势

### 为什么双优股票是黄金标的？

双优股票兼具：
- **安全边际**：估值低、超跌明显，下行空间有限
- **上涨动能**：趋势强劲、动量充足，上涨概率高
- **风险收益比**：进可攻（趋势向上）、退可守（估值支撑）

### 推荐列表中的处理

**重复推荐策略**：
- 双优股票会在推荐列表中**重复出现两次**（超跌榜一次 + 趋势榜一次）
- 在输出中用 **⭐双优** 标签着重标注
- 在报告顶部单独列出双优股票专区，方便快速识别

**示例输出**：
```
⭐⭐⭐ 双优股票（既超跌又趋势强，黄金标的，重点关注！）
| 代码 | 名称 | 价格 | MR得分 | 趋势得分 | 超跌榜排名 | 趋势榜排名 | 说明 |
| 300750 | 宁德时代 | 215.30 | 8.5 | +0.72 | 第3 | 第2 | 低估值+强趋势 |
```

## 🚀 v5.2 完整趋势引擎（Phase 1 & 2，2026-03-26）

### 4层因子体系

**背景**: 原趋势榜过于简单，仅基于 `Trend_Composite`，缺少技术确认、量价配合、相对强度等关键因子。

**改进**: 构建完整的4层因子体系：

#### 第1层：基础趋势（40%）
- `Trend_Composite`: ADX强度 + 均线排列 + 波动率调整动量
- `Momentum_Adj`: 波动率调整动量

#### 第2层：技术确认（30%）
- **MA金叉/死叉**: MA5 上穿/下穿 MA20
- **MACD金叉/死叉**: DIF 上穿/下穿 DEA
- **布林带位置**: 价格在布林带中的位置（%B）

#### 第3层：相对强度（20%）
- **个股 vs 指数**: 使用信息比率计算超额收益
- **个股 vs 行业**: 使用信息比率计算行业内相对表现

#### 第4层：量价配合（10%）
- **放量上涨** → +1.0（强多）
- **放量下跌** → -1.0（强空）
- **缩量上涨** → +0.3（弱多）
- **缩量下跌** → -0.3（弱空）

### 趋势质量得分公式

```python
base_trend = 0.7 * trend_score + 0.3 * momentum_score

trend_rank_score = (0.4 * base_trend +           # 基础趋势 40%
                   0.3 * tech_confirm_score +    # 技术确认 30%
                   0.2 * relative_strength_score + # 相对强度 20%
                   0.1 * volume_confirm_score)   # 量价配合 10%
```

### 回测结果（200只股票）

**超跌榜**:
- 平均收益率: **0.43%**（每5日）
- 胜率: **51.30%** ✅
- Sharpe: **0.101** ✅
- 特点：稳健，风险调整后收益优秀

**趋势榜**:
- 平均收益率: **0.50%**（每5日）✅
- 胜率: **47.57%**
- Sharpe: **0.065**
- 特点：收益高，但波动大

**结论**: 双引擎互补，超跌榜稳健，趋势榜进攻性强。
