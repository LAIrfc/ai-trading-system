## 一、情绪因子策略（1 个）✅ 已实现（V3.3）

### 市场情绪指标（Sentiment）

| 项目 | 说明 |
|------|------|
| **名称** | 市场情绪指标（Sentiment） |
| **描述** | V3.3：多指标 Z-score 合成 S，60 日 20/80 分位 S_low/S_high；极端+次日确认+个股趋势过滤（MACD 柱斜率、5 日高低、ADX>25）。 |
| **最小 K 线数** | 65 条（满足趋势过滤计算） |

**核心原理：**

- 多指标 60 日 Z-score、缺项权重 0 并归一化，合成 S；S_low/S_high 为 60 日滚动 20/80 分位。当前实现含沪深300 动量+HV20，可扩展涨跌家数比、换手率、两融等。

**信号规则（V3.3）：**

- **买入**：前一日 S < S_low 且当日 S 回升，且个股趋势过滤通过（MACD 柱 3 日斜率<0、创 5 日新低、ADX>25）。
- **卖出**：前一日 S > S_high 且当日 S 回落，且趋势过滤通过（斜率<0、创 5 日新高、ADX>25）。
- 无 V3.3 数据时回退为旧版 0~100 单指标逻辑（无趋势过滤）。

## 二、消息面策略（1~2 个）

### 1. 个股新闻情感策略（NewsSentiment）✅ 已实现

| 项目 | 说明 |
|------|------|
| **名称** | 新闻情感分析（NewsSentiment） |
| **描述** | 基于财经新闻、公告的 NLP 情感分析，识别正面/负面事件，结合事件影响力产生信号。 |
| **最小 K 线数** | 0（事件驱动，无需 K 线） |

**核心原理：**

- 实时爬取或接入财经新闻 API，使用预训练情感模型对标题和摘要打分（-1 到 1）。
- 累积情感分数：当连续 N 篇新闻情感均值 **超过阈值（如 0.3）** 且当前价格未大幅偏离 → **买入**；**低于阈值（如 -0.3）** → **卖出**。
- 考虑事件影响力：结合新闻来源权威性、阅读量等加权。

## 三、政策面策略（1 个）✅ 已实现

### 政策事件驱动（PolicyEvent）

| 项目 | 说明 |
|------|------|
| **名称** | 政策事件驱动（PolicyEvent） |
| **描述** | 识别宏观政策、行业政策发布，判断利好/利空方向，并结合市场反应强度生成信号。 |
| **最小 K 线数** | 0 |

**核心原理：**

- 建立政策关键词库（如「降准」「减税」「产业规划」等），通过新闻源监测政策发布。
- 对每条政策进行人工或半自动标签（利好/利空/中性），并赋予影响力评分（如国家级 > 部委级）。

**信号生成：**

- 重大 **利好** 政策发布后，若市场未过度透支（如涨幅 < 3%）→ **买入信号**
- 重大 **利空** 政策发布后 → **卖出信号**
- 影响力较小的政策可仅作为辅助因子（调整置信度）

## 四、融入现有组合策略的方式

现有组合策略（保守、均衡、激进）基于多个底层策略的投票结果。新策略可像技术面、基本面一样独立投票，并参与组合计算。

### 方式 1：直接加入投票池

- 将情绪、消息、政策策略加入策略列表，总数从 8 个增至 **11~12 个**。
- 每个策略仍输出「买入/卖出/持有」及置信度，组合逻辑不变：
  - **保守组合**：多数看多才买入（如 >70%），少数看空即卖出（如 >30% 看空）。
  - **均衡组合**：过半策略同意就行动（买入 >50%，卖出 >50%）。
  - **激进组合**：加权投票，持有不计分；可给新策略更高权重（如政策面 1.5，消息面 1.2）。

### 方式 2：作为条件过滤器

- 将情绪、政策因子作为 **前置条件**，只有满足条件时才启用其他策略。例如：
  - 情绪指数处于 **非极端区间** 时，才考虑技术面信号。
  - 政策面有 **重大利好** 时，对所有买入信号进行「强化」（如提高置信度）。
- 需在组合策略中增加条件判断层。

### 方式 3：动态权重调整

- 根据市场状态（震荡市/趋势市）动态调整各策略权重：
  - 消息密集期（如财报季）→ 提高新闻策略权重。
  - 政策敏感期 → 提高政策策略权重。
- 可在组合层实现权重动态分配。

## 五、实施注意事项

| 事项 | 说明 |
|------|------|
| **数据源** | 情绪因子需多源数据（行情、两融、期权等）；新闻需接入专业财经 API 或自行爬取；政策需建立标签库，初期可人工标注，后期尝试 NLP 分类。 |
| **信号时效性** | 消息、政策多为事件驱动，信号持续时间短；建议在组合中设置 **信号有效期（如 24 小时）**，过期转为持有。 |
| **回测验证** | 新策略需 **单独回测**，确认有效后再加入组合，避免过拟合。 |
| **置信度校准** | 新策略的置信度可基于历史胜率或事件强度设定，确保与现有策略可比。 |
