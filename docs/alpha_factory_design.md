## 📊 核心模块说明

### 1. IC分析器（ICAnalyzer）

**文件**: `tools/analysis/ic_analyzer.py`

**功能**:
- 计算策略信号与未来收益的Spearman相关系数
- 评估策略有效性（IC均值、IR、胜率）
- 分析策略之间的相关性
- 生成权重优化建议

**关键指标**:
- **IC (Information Coefficient)**: 因子值与未来收益的相关系数
  - IC > 0.05: 优秀
  - IC > 0.02: 有效
  - IC < 0.02: 无效
- **IR (Information Ratio)**: IC均值 / IC标准差，衡量稳定性
  - IR > 0.8: 优秀
  - IR > 0.5: 良好
  - IR < 0.3: 不稳定

**用法**:
```bash
python3 tools/analysis/ic_analyzer.py \
    --pool mydate/stock_pool_all.json \
    --count 100 \
    --days 500 \
    --forward 5 \
    --workers 8 \
    --skip-ai
```

### 2. 市场状态识别引擎（MarketRegimeEngine）

**文件**: `src/core/market_regime.py`

**功能**:
- 基于沪深300指数判断市场状态
- 状态分类：bull（牛市）、bear（熊市）、sideways（震荡）
- 历史状态分析和统计

**判断规则**:
```python
# 牛市：短期均线 > 长期均线 且 波动率 < 历史均值
if MA20 > MA200 and VOL20 < VOL_MEAN:
    regime = 'bull'

# 熊市：短期均线 < 长期均线 且 波动率 > 历史均值
elif MA20 < MA200 and VOL20 > VOL_MEAN:
    regime = 'bear'

# 震荡：其他情况
else:
    regime = 'sideways'
```

**用法**:
```python
from src.core.market_regime import MarketRegimeEngine

engine = MarketRegimeEngine()
current_regime = engine.get_current_regime()  # 'bull', 'bear', 'sideways'
stats = engine.get_regime_stats()
```

## 📝 技术要点

### IC计算的正确性

**关键修正**：
- BUY信号 → 因子值 = +confidence（预期涨）
- SELL信号 → 因子值 = -confidence（预期跌）
- IC = Spearman(因子值, 未来收益)

这样：
- 策略预测涨且真涨 → IC为正 ✅
- 策略预测跌且真跌 → IC为正 ✅
- 策略预测涨但真跌 → IC为负 ❌
- 策略预测跌但真涨 → IC为负 ❌

### 避免未来数据泄露

1. **IC计算**: 使用T日信号预测T+5日收益（合理）
2. **权重更新**: 只使用截至T日的历史数据
3. **回测**: 严格T日信号 → T+1日开盘执行

### 性能优化

1. **并行处理**: ThreadPoolExecutor并行分析多只股票
2. **缓存机制**: 指数数据、基本面数据当天缓存
3. **跳过AI策略**: IC分析时跳过需要AI调用的策略（PolicyEvent, NewsSentiment）

- **L2层**: 权重动态调整

**验证结果** (2026-03-24):
- 测试了1296个动态权重配置（30只股票，2023-2026年）
- **所有动态配置在测试期均劣于固定权重**
  - 固定权重：夏普0.118，年化5.33%，回撤5.98%
  - 最优动态：夏普0.078，年化5.06%，回撤6.00%
- **结论**: 当前固定权重已是最优方案，L2层动态调整无法提升表现

**不启用原因**:
1. 市场状态识别滞后（MA20/MA200滞后1-2个月）
2. 训练期最优系数在测试期失效（过拟合）
3. 固定权重已经过多轮优化（v3/v4），难以进一步提升
