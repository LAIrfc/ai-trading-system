## 📚 技术细节

### 因子正交化顺序

```python
factor_order = [
    'relative_strength',  # 最独立，作为基准
    'base_trend',         # 最重要
    'tech_confirm',       # 次之
    'volume_confirm'      # 最后
]
```

### 动态权重公式

```python
# 趋势市权重
w_trend = [0.5, 0.3, 0.1, 0.1]

# 震荡市权重
w_range = [0.3, 0.3, 0.2, 0.2]

# 线性插值
alpha = (regime_score + 1) / 2
weights = alpha * w_trend + (1 - alpha) * w_range
```

### 门控交互

```python
# 仅当base_trend > 0时启用
if base_trend_orth > 0:
    interaction = base_trend_orth * volume_confirm_orth * 0.1
    score += interaction
```
